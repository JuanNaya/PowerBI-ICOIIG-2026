## 🗂️ Proyecto: Análisis de Ventas Contoso

### ¿Qué es Contoso?

**Contoso** es una empresa ficticia de electrónica de consumo creada por Microsoft para demos y formación. En este curso trabajaremos con datos de:

- 🛍️ **Ventas** por producto, tienda y canal
- 📅 **Períodos temporales** (años, trimestres, meses)
- 🏪 **Tiendas** físicas y online
- 📦 **Productos** de diferentes categorías
- 👥 **Clientes** y segmentación

### Objetivo del Proyecto

Crear un dashboard interactivo que permita a la dirección de Contoso:
- Analizar ventas por producto, tienda y período
- Identificar tendencias y patrones
- Tomar decisiones estratégicas basadas en datos

## 📚 Contenidos por Sesión

### 📅 Sesión 1 - Viernes 10 abril (16:00-20:00h)
**Primeros pasos con Power BI**
- Instalación y configuración de Power BI Desktop
- Conexión a fuentes de datos (Excel)
- Creación del modelo de datos básico
- Primeras métricas y visualizaciones
- Introducción a la interfaz

### 📅 Sesión 2 - Sábado 11 abril (09:00-13:00h)
**Visualizaciones y Elementos Visuales**
- Tipos de visuales: tablas, matrices, tarjetas
- Gráficos de columnas, barras, líneas y áreas
- Configuración de propiedades: título, tamaño, posición
- Tooltips personalizados
- Filtros y segmentaciones

### 📅 Sesión 3 - Viernes 17 abril (16:00-20:00h)
**Diseño y Navegación Avanzada**
- Principios de diseño de dashboards
- Temas y paletas de colores
- Marcadores y navegación entre páginas
- Botones interactivos
- Drill through y drill down
- Parámetros de campo

### 📅 Sesión 4 - Sábado 18 abril (09:00-13:00h)
**Publicación y Compartir**
- Diseño de vista móvil
- Optimización del informe
